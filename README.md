# G9-LATAM-TEAM-50-FinanceAI
📊 Finance AI – Asistente Inteligente de Salud Financiera

Solución inteligente capaz de analizar el comportamiento financiero de un usuario a partir de sus transacciones e información financiera, generando una visión más completa de su salud financiera.
La solución deberá recibir información relacionada con gastos y hábitos financieros, como descripción de transacciones, montos, categorías de gastos, ingresos mensuales, frecuencia de ahorro, nivel de endeudamiento y otros indicadores relevantes.
